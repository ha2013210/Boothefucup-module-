# Boothefucup-module-
It's a magisk module that replaces any bootanimation without manually put in /system/product/media (very useful for read only partitions or legacy devices)
extract the zip file
Put the bootanimation in boothefucup/system/product/media
Select multi select (or 3 dots if mobile user) select both system and module.prop and compress .zip and set to no compression 
Zarchiver is recommended and make sure you are rooted it's a module duh
it might work on kernel based root managers like kernelsu
